**<h2 style="text-align: Left;"><u>Use Case Overview</u></h2>**


This workflow will orchestrate AWS Step Functions to run a job to print out Hello World


**<h2 style="text-align: Left;"><u>Use Case Technical Discusson</u></h2>**


This workflow will start by orchestrating the Step funciton Service utilizing the sttate machine <strong>mqn-state-machine</strong> to execute <strong>ani-state-machine</strong>.   This will execute 2 steps.  First one will be "<strong>Hello</strong> and the second will be <strong>World</strong>.

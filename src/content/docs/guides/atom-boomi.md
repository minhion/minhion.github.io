# Boomi AtomSphere

**Boomi AtomSphere** is an on-demand multi-tenant cloud iPaaS for connecting cloud and on-premises applications and data. The platform enables customers to design cloud-based integration processes called Atoms and transfer data between cloud and on-premises applications.

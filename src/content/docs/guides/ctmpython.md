# Use Case Overview 
The Control-M Python Client is a python library that can be used to define and run workflows in Control-M. This client will be released as an open-source offering to the public on <b>12/1/2021</b>. This use case will create a VS Code Server instance that is web accessible with the Control-M Python Client embedded and ready to use.  

## Use Case Technical Explanation 

To view the demo flow code-base please navigate to the [CTM Python Client](https://ctm-git.trybmc.com/automated-demos/ctm-python-client)

# Azure Logic Apps (via Azure Job)

**Azure Logic Apps** is a cloud-based platform that lets you create and run automated workflows with little to no code. Using the visual designer and selecting from prebuilt operations, you can build a workflow that integrates and manages your apps, data, services, and systems.

- **[Azure Logic Apps Additional Information](https://azure.microsoft.com/en-us/products/logic-apps/#overview)**
